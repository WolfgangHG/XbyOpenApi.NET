This package provides OAuth2 authentication for the .NET client "XbyOpenApi.Core" for the X / Twitter REST api.

It contains no UI, just common code for handling OAuth2, so you either have to build or own UI or if you use WinForms,
you can use the package "XbyOpenApi.OAuth2.WinForms"

For details about OAuth2 for X, see https://github.com/WolfgangHG/XbyOpenApi.NET/blob/main/doc/OAuth2.md
