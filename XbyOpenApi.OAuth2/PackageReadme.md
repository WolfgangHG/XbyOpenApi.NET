This package provides ui-agnostic helpers for OAuth2 authentication for the .NET client [XbyOpenApi.Core](https://www.nuget.org/packages/XbyOpenApi.Core) for the X / Twitter REST api.

It contains no UI, just common code for handling OAuth2, so you either have to build or own UI or in a WinForms environment,
you can use the package [XbyOpenApi.OAuth2.WinForms](https://www.nuget.org/packages/XbyOpenApi.OAuth2.WinForms)

See usage doc at https://github.com/WolfgangHG/XbyOpenApi.NET

For details about OAuth2 for X, see https://github.com/WolfgangHG/XbyOpenApi.NET/blob/main/doc/OAuth2.md
