This package provides ui-agnostic helpers for OAuth1 authentication for the .NET client [XbyOpenApi.Core](https://www.nuget.org/packages/XbyOpenApi.Core) for the X / Twitter REST api.

OAuth1 handling is done by using [TinyOAuth1](https://github.com/johot/TinyOAuth1)

It contains no UI, just common code for handling OAuth1, so you either have to build or own UI or in a WinForms environment,
you can use the package [XbyOpenApi.OAuth1.WinForms](https://www.nuget.org/packages/XbyOpenApi.OAuth1.WinForms).

See usage doc at https://github.com/WolfgangHG/XbyOpenApi.NET

For details about OAuth1 for X, see https://github.com/WolfgangHG/XbyOpenApi.NET/blob/main/doc/OAuth1.md
