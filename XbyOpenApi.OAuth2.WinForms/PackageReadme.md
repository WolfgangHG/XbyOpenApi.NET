This package provides WinForms components for OAuth2 authentication for the .NET client 
[XbyOpenApi.Core](https://www.nuget.org/packages/XbyOpenApi.Core) for the X / Twitter REST api.

With use of the package "Microsoft.Web.WebView2", you can show a login form to acquire an OAuth2 Access Token.

See usage doc at https://github.com/WolfgangHG/XbyOpenApi.NET
